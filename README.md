# Multi-Threaded Web Server

A Java-based project demonstrating different web server architectures including Single-Threaded, Multi-Threaded, and Thread Pool implementations.

## Technologies

* Java
* Socket Programming
* Multithreading
* Thread Pools

## Features

* Single-threaded server implementation
* Multi-threaded server implementation
* Thread pool based request handling
* Client-server communication using TCP sockets

## Learning Outcomes

* Concurrent programming
* Thread management
* Socket programming
* Server scalability concepts
